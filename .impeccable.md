## Design Context

### Users
Residents of Taif (men and women), ages 18-65, interested in health, sports, and walking in nature. Varying fitness levels from beginners to advanced. They browse during daytime breaks, often on mobile, looking for walking schedules, trail info, or event registration. Many are not tech-savvy — clarity beats cleverness.

### Brand Personality
Warm, community-driven, motivating. Feels like a friendly local sports club — not a corporate entity. Authentically Saudi, rooted in Taif's mountain character (mountains, roses, fresh air). First walking association in Saudi Arabia.

**Tone**: organic/natural, welcoming. Like a neighbor inviting you on a morning walk — warm, encouraging, never pushy.

**3 brand words**: warm, community-driven, motivating

### Aesthetic Direction
- **Light mode primary** (users browse during daytime), dark mode as secondary option
- **Organic warmth** — warm-tinted backgrounds, never cold/sterile
- **Natural textures** — subtle grain, soft shadows, earthy greens
- **Community photography** — real people walking together, not stock fitness models
- **NOT**: glassmorphism, gradient text, neon glows, corporate blue, techy aesthetics
- **Anti-references**: SaaS dashboards, tech startup landing pages, fintech apps

### Design Principles

1. **Warmth over sterility** — Every surface should feel welcoming. Use warm-tinted whites and grays (never pure #fff or #000). Shadows should be soft and warm, not cold blue-gray.

2. **Community-first hierarchy** — Lead with people, not features. Photos of group walks beat abstract icons. Testimonial quotes beat marketing copy. Show the real community.

3. **Natural rhythm** — Break the grid monotony. Stagger cards, vary sizes, let content breathe asymmetrically. Like a mountain trail — not every step is the same size.

4. **Clarity beats cleverness** — Users range from 18 to 65, many not tech-savvy. Big readable text, obvious CTAs, clear navigation. Every label should make sense without thinking.

5. **Authentically Saudi** — Arabic RTL as first-class, not afterthought. Saudi dialect for casual text, formal Arabic for official content. Cultural references that resonate locally.

## Implementation Rules

### Color System

```css
/* Primary brand — Taif's green mountains and rose gardens */
--color-brand: #00a651;
--color-brand-light: #33b874;
--color-brand-warm: #00964a;

/* Text — warm charcoal, never pure black */
--color-charcoal: #1c1f24;
--color-charcoal-light: #2c3038;

/* Surfaces — warm-tinted, never pure white */
--color-warm-white: #fafaf7;
--color-warm-gray: #f2f1ec;
--color-surface: #f7f8f5;
--color-surface-dark: #e8eae3;
```

**Rules**:
- Body background: `warm-white` (#fafaf7), never #fff
- Alternate sections: `warm-gray` (#f2f1ec)
- Text: `charcoal` (#1c1f24), never #000 or #1a1a1a
- Accent shadows: use brand green tint, never cold blue

### Typography

**Single font**: Cairo — used for all text (headings + body). Weight variation (400/600/700) creates hierarchy.

```css
.font-display { font-family: var(--font-cairo), 'Cairo', sans-serif; }
.font-body    { font-family: var(--font-cairo), 'Cairo', sans-serif; }
```

**Heading hierarchy**:
- h1: `text-4xl md:text-6xl font-bold` — `.font-display`
- h2: `text-3xl md:text-4xl font-bold` — `.font-display`
- h3: `text-xl font-bold` — `.font-display`
- Body: `text-base md:text-lg` — Cairo (default)

### Spacing & Layout

- **Section padding**: `py-16 md:py-20` (not py-24 or py-32 — avoid over-stretching)
- **Max width**: `max-w-7xl` for content, `max-w-4xl` for text-heavy sections
- **Card gaps**: `gap-6` standard, `gap-4` compact
- **Internal card padding**: `p-6 md:p-8`
- **Headings**: Left-aligned (RTL: `text-right`), not centered — creates visual anchor
- **Break monotony**: Stagger cards with `mt-0`, `mt-6`, `mt-3` etc.

### Anti-Patterns (NEVER use)

- `backdrop-blur` / glassmorphism / frosted glass effects
- Gradient text (`bg-gradient-to-r bg-clip-text text-transparent`)
- Glow shadows (`shadow-[0_0_40px_rgba(0,166,81,0.3)]`)
- `rounded-full` on large buttons (use `rounded-xl` instead)
- Uniform card grids with identical sizing (vary heights, stagger)
- Centered everything — mix left-aligned headings with centered content
- Pure `#fff` background or `#000` text
- `text-white` drop-shadow effects on hero text

### Accessibility

- Every interactive element needs `focus-visible` outline (2px solid brand, offset 2px)
- All images need `alt` text in Arabic
- Use `aria-label` on sections, `aria-expanded` on toggles
- `role` attributes on landmark elements
- Respect `prefers-reduced-motion` — disable animations
- Skip-to-content link as first focusable element
- Color contrast: minimum 4.5:1 for body text, 3:1 for large text
- Touch targets: minimum 44x44px on mobile

### Component Patterns

- **Buttons**: `rounded-xl`, never `rounded-full`. Solid fills, not glass.
- **Cards**: `rounded-2xl`, `border border-charcoal/5`, soft shadow on hover only
- **Sections**: Alternate `bg-warm-white` / `bg-warm-gray` for visual separation
- **Icons**: `lucide-react` with `strokeWidth={1.5}`, inside rounded containers
- **Images**: Next.js `<Image>` with `sizes` attribute, `object-cover`, subtle hover scale
- **Animations**: Framer Motion `whileInView` with `viewport={{ once: true }}`, subtle y-translates (20-30px), not dramatic
